# YOLOv5 Underwater Marine Object Detection

This Streamlit app utilizes the YOLOv5 model for detecting underwater marine objects in images and videos. Users can upload their files, and the app will process and display the detection results.

## Requirements

- Python 3.7+
- Streamlit
- PyTorch
- OpenCV
- Pillow

## Usage

1. Upload an image or video.
2. The model will perform object detection.
3. Download the processed video with detected objects (if applicable).
